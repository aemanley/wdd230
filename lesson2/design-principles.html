White Space and Clean Design

Moz

moz.com

This web page use a lot of empty or negative space around the content.
Things are not overlapping or to close to each other,
we can easily scroll through the page, text is easy to read.

Contrast

Deezer

deezer.com

The contrast that the background color makes with the colors used in the squares makes the page quite striking and helps us to better navigate the page since we can notice each element separately thanks to the color contrast.

Repetition

Target

target.com

This web page reuses same or similar elements, it reuses colors, and shapes that are used as holders of elements.
